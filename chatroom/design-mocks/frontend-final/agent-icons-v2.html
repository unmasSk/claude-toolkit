Design Notes

  • All icons: 24x24 viewBox, stroke-width 2, round caps/joins, fill none
  • Paths sourced from / inspired by Lucide icon library (MIT license)
  • Animations: pure SMIL (<animate> + <animateTransform>) -- no CSS, no JS
  • Animation durations: 1.2s - 4s, subtle and non-distracting
  • Scales cleanly from 16px to 64px due to viewBox-based rendering
  • Three color states: active (#c0c0c0), inactive (#666), disconnected (#3a3a3a)
  • Animations only on active state -- inactive and disconnected are static

V3 Design Notes

  • All animations use CSS @keyframes -- no SMIL, no JavaScript
  • Primary easing: cubic-bezier(0.25, 1, 0.5, 1) (quart-out) for natural deceleration
  • Overshoot easing: cubic-bezier(0.34, 1.56, 0.64, 1) (back-out) for pop/settle effects
  • Continuous loops (Argus sweep) use linear timing
  • Idle loop durations: 2.5s - 4s for organic, non-distracting motion
  • Transform chains: scale + rotate + translate combined in single keyframe sequences
  • Offset-path used for House ECG trailing dots (motion along SVG path)
  • Same SVG paths as V2 -- drop-in replacement, only animation layer changes
